    • How do for loops and while loops differ in their usage and syntax?

  • Initialization is Within the loop header for for loop and Outside the loop for while loop
  • Condition is Within the loop header for for loop and Before the loop body for while loop
  • Increment/Decrement is done Within the loop header for for loop and Within the loop body for while loop
