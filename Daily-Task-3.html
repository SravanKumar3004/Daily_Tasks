Global and Local Scope

  • globalVar is declared outside of a function, making it a global variable.
  • It can be accessed from anywhere in the code

  • localVar is declared inside the scopeTest function, making it a local variable.
  • It can only be accessed within the scopeTest function.
  • While trying to access localVar outside of scopeTest will result in a ReferenceError because localVar is not defined in the global scope.
